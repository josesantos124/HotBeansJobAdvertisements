# Hot Beans Job Advertisements
Hot Beans Job Advertisements is a front-end recruitment website developed as part of BTEC IT coursework.
The project focuses on creating a clean, responsive user interface that allows recently graduated web developers to browse job opportunities.

The website was built using HTML, CSS, and JavaScript, with an emphasis on:

Responsive design across different screen sizes

Clear layout and visual hierarchy

User-friendly navigation and accessibility

Front-end development best practices
